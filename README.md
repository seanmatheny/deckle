# deckle

A native macOS notebook app that resembles a traditional notebook, backed by a local PDF folder structure.

## Development

This repository is set up as a Swift Package:

- `DeckleCore`: shared models/indexing logic
- `DeckleApp`: native SwiftUI macOS app target

Run tests:

```bash
swift test
```

Run app on macOS:

```bash
swift run Deckle
```
